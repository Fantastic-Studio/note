# note

这是一个用来写作的小工具，基于JavaFX，以及SpringBoot，内部采用H2数据库进行存储，可以导出为HTML格式。

当前版本已经可以正常使用了，纵版UI效果比较好，如果是Mac系统的话，中文输入可能会有些输入法的bug。

目前markdown采用flex-marker库进行解析，可以支持大多数markdown语法。
